## Introduction
A simple qwik-city application to showcase the stripe integration
## Steps
1. Obtain a stripe key from stripe account(https://stripe.com/)
2. Rename example.env to .env.local
3. Add stripe key and strie webhook scerect
4. npm i
5. npm start
